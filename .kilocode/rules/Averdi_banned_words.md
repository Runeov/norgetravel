# THE WAR ON BUREAUCRACY

Averdi fights "kansellistil" (officialese) — the dusty, passive, noun-heavy language that plagues Norwegian professional communication.

**Rule:** AI and human writers must run a "Search and Replace" pass on all text.

## THE NOMINALIZATION WAR

### The Disease: Substantivsyken
Bureaucratic Norwegian turns verbs into nouns, sucking life from sentences.

**Rule:** If you can DO it, use a VERB.

### Common Offenders

| ❌ Noun Form (Avoid) | ✅ Verb Form (Use) |
|----------------------|-------------------|
| Utførelse av | Å utføre / Utfører |
| Gjennomføring av | Å gjennomføre / Gjennomfører |
| Vurdering av | Å vurdere / Vurderer |
| Behandling av | Å behandle / Behandler |
| Fastsettelse av | Å fastsette / Fastsetter |
| Foretakelse av | Å foreta / Foretar |
| Iverksettelse av | Å iverksette / Starter |

### Example Transformation

```
❌ BUREAUCRATIC:
"Utførelse av søknadsskriving til Sametinget kan være krevende for foretak uten erfaring."

✅ AVERDI (Good):
"Å skrive søknad til Sametinget er krevende hvis du mangler erfaring."

✅ AVERDI (Better - Active):
"Sliter du med søknaden til Sametinget? Vi vet hvordan du knekker koden."
```

## PASSIVE VOICE ELIMINATION

### The Rule
Use **active voice in 90% of sentences**.

### Transformation Examples

| ❌ Passive (Avoid) | ✅ Active (Use) |
|-------------------|-----------------|
| Det gjøres oppmerksom på at fristen er endret | **Merk deg den nye fristen** |
| Det kan søkes om tilskudd | **Du kan søke om tilskudd** |
| Søknaden ble mottatt | **Vi mottok søknaden** |
| Det ble besluttet at | **Vi bestemte at** |
| Regnskapet skal leveres innen | **Lever regnskapet innen** |

### Strategic Exception
Use passive voice ONLY when:
1. You want to soften a blow: "Søknaden ble avslått" (softer than "Sametinget avslo søknaden")
2. The actor is truly irrelevant

## FILLER PHRASES TO DELETE

### Completely Remove These

| ❌ Delete | Why |
|-----------|-----|
| Det gjøres oppmerksom på at | Just say the thing |
| Det må anses at | Adds nothing |
| Det bemerkes at | Just remark it directly |
| Med hensyn til | Say "om" instead |
| I forbindelse med | Say "ved" or "når" |
| For så vidt angår | Just say "for" |

### Example

```
❌ BEFORE:
"Det gjøres oppmerksom på at det med hensyn til den nye forskriften må anses at fristen er endret."

✅ AFTER:
"Fristen er endret."
```

### Check Filler
